# Tools List

This document contains a comprehensive list of all tools available in the convert-all application, compiled from the registry files.

## Unit Conversions

- **Length Converter**: Convert between microns, millimeters, inches, feet, meters, kilometers, miles
- **Mass & Weight Converter**: Convert between micrograms, grams, ounces, pounds, kilograms, tons
- **Temperature Converter**: Convert between Celsius, Fahrenheit, Kelvin, and Rankine
- **Volume Converter**: Convert between milliliters, cups, pints, liters, gallons, cubic meters
- **Currency Converter**: Convert between USD, EUR, GBP, Bitcoin, Ethereum with live rates
- **Time Converter**: Convert between nanoseconds, seconds, minutes, hours, days, years
- **Speed Converter**: Convert between mph, km/h, m/s, knots, Mach, speed of light
- **Area Converter**: Convert between square feet, square meters, acres, hectares
- **Energy Converter**: Convert between joules, calories, BTU, kWh, electron volts
- **Pressure Converter**: Convert between PSI, Bar, Pascal, and other pressure units
- **Data Size Converter**: Convert between KB, MB, GB, TB, and other data units
- **Spacetime Converter (Lorentz Transformation)**: Convert relativistic space-time coordinates using Lorentz transformations

## Calculators

- **Graphing Calculator**: Plot mathematical functions with symbolic expressions
- **Scientific Calculator**: Advanced calculator with scientific functions
- **BMI Calculator**: Calculate your Body Mass Index
- **Interest, Loan and Mortgage Calculator**: Calculate interest rates, loan payments, mortgages, and total interest
- **Tip Calculator**: Calculate tips and split bills
- **Percentage Calculator**: Calculate percentages, increases, and decreases
- **Date Calculator**: Add days to dates and calculate date differences
- **Calorie Calculator**: Calculate daily calorie needs based on age, gender, and activity level
- **Protein Calculator**: Calculate daily protein needs based on weight and activity level
- **Pregnancy Due Date Calculator**: Calculate due date and weeks pregnant from last period
- **Paint Calculator**: Calculate gallons of paint needed for room dimensions
- **Flooring Calculator**: Calculate square footage and materials needed for flooring
- **Concrete Calculator**: Calculate cubic yards of concrete needed for projects

## File Converters

- **Document Converter**: Convert between PDF, Word, Text, and other document formats
- **Spreadsheet Converter**: Convert between Excel, CSV, and spreadsheet formats
- **Data Format Converter**: Convert between JSON, XML, YAML, and CSV data formats
- **Base64 Encoder/Decoder**: Encode files to Base64 or decode Base64 to files
- **Archive Tools**: Create ZIP archives or extract files from archives

## Media Converters

- **Image Converter**: Convert between JPG, PNG, WebP, GIF, and other image formats
- **Audio Converter**: Convert between MP3, WAV, FLAC, AAC, and other audio formats
- **Video Converter**: Convert between MP4, AVI, MKV, MOV, and other video formats

## Developer Tools

- **JSON Formatter**: Format, validate, and beautify JSON data
- **Base64 Encoder/Decoder**: Encode and decode Base64 strings
- **URL Encoder/Decoder**: Encode and decode URL strings
- **Hash Generator**: Generate MD5, SHA-1, SHA-256, and SHA-512 hashes
- **UUID Generator**: Generate UUIDs (v4) and GUIDs
- **Timestamp Converter**: Convert between Unix timestamps and human-readable dates
- **Regex Tester**: Test and debug regular expressions
- **JWT Decoder**: Decode and inspect JSON Web Tokens
- **XML Formatter**: Format and prettify XML data
- **CSV Formatter**: Clean and format CSV data
- **Text Case Converter**: Convert text between different cases
- **Email Extractor**: Extract email addresses from text
- **URL Extractor**: Extract URLs from text
- **JSON Minifier**: Compress JSON by removing whitespace
- **JSON Validator**: Validate JSON syntax and structure

- **Text Processor & Data Tools**: Extract data, format code, and transform text in one tool
- **Audio Equalizer**: Adjust audio frequencies with EQ
